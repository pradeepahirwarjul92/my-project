Q1 - Git Workflow 
Ans - Working Directory (Your Workspace)→ Staging Area (Preparation Zone) → Repository

	- Working Directory (Your Workspace)
		Example:
			You edit Employee.java → changes are only in your local file
	
	- Staging Area (Preparation Zone)
		Example:
			git add Employee.java
	
	- Repository (Permanent Storage)
		Example:
			git commit -m "Added salary calculation logic"
