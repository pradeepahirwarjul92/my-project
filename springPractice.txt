Q2 - Connect multiple data source 
Ans - In Spring Boot multi-datasource setup, each database requires a DataSource for connection, 
	EntityManagerFactory for ORM mapping, and TransactionManager for transaction handling. These are linked to repositories using @EnableJpaRepositories.
Ans - Each DB needs 3 things: Connection + Mapping + Transaction.”
1. DataSource        → Connection (DB se connect)
2. EntityManager     → Mapping (Java ↔ Table)
3. TransactionManager → Control (commit / rollback)
