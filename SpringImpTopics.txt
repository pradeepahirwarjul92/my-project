3. Spring Cloud Components (VERY IMPORTANT) ⭐

These are core microservice tools.

1️⃣ Service Discovery

Service registers itself so other services can find it.

Common Tool:

Netflix Eureka

Architecture

Service A → Eureka Server
Service B → Eureka Server

Service A → call Service B via Eureka

Annotations

@EnableEurekaServer
@EnableDiscoveryClient

Common Tool:

5. Fault Tolerance / Resilience

Microservices must handle failures.

Tool

Resilience4j

Patterns

Circuit Breaker
Service A → Service B

If B fails repeatedly
Circuit Breaker → Stop calling B

Annotation

@CircuitBreaker(name="paymentService")
